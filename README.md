## ğŸš€ Quick Start

### Prerequisites

- Node.js 18+ installed
- npm or yarn package manager

### Installation

```bash
# Clone the repository
git clone https://github.com/ChulkovDanila/readme-template-generator.git

# Navigate to project directory
cd readme-template-generator

# Install dependencies
npm install

# Start development server
npm run dev
```

Visit `http://localhost:5173` in your browser!

## ğŸ¯ Usage

1. **Choose a Template** - Select from the left sidebar
2. **Edit Placeholders** - Click on highlighted text to edit (e.g., `[Your Name]`)
3. **Configure GitHub** - Click the GitHub button to set your username
4. **Customize** - Use the toolbar to format text, add links, images, etc.
5. **Export** - Copy to clipboard or download as `README.md`
